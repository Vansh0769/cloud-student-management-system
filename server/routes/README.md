# ☁️ Cloud-Based Student Management System

A simple full-stack CRUD application developed as a **Microsoft Azure Developer (AZ-204)** minor project.

---

## 📌 Features

- View Students
- Add Student
- Update Student
- Delete Student
- Search Students
- Responsive Dashboard
- REST API Integration

---

## 🛠️ Tech Stack

### Frontend
- React
- Vite
- Bootstrap
- Axios

### Backend
- Node.js
- Express.js

### Database
- JSON File (Temporary)

### Cloud (Future Deployment)
- Azure Static Web Apps
- Azure App Service
- Azure SQL Database
- Azure Blob Storage

---

## 📂 Project Structure

```
client/
server/
```

---

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/cloud-student-management-system.git
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run dev
```
